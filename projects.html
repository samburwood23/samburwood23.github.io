Projects

A collection of systems, experiments, and conceptual tools exploring AI agents, reasoning, complexity theory, and human–AI interaction.

  • Technical Documentation System — A structured, generative documentation pipeline designed for complex AI projects. It automates index generation, semantic linking, and topic navigation across multi-module repositories, enabling maintainable and scalable technical knowledge.
  • Stateful vs Stateless Systems — A set of technical explorations on how memory, context windows, and temporal continuity shape AI agent behaviour. The project examines architecture trade-offs, long-term vs short-term reasoning, human–AI temporal mismatch, and how to design agents that act as stable collaborative partners rather than isolated, stateless tools.
  • Adaptive Fuzzy Logic Controller — A dynamic fuzzy-rule decision system that reasons under uncertainty. It blends graded logic with adaptable inference rules, allowing smoother and more interpretable reasoning than traditional crisp decision systems.
  • Complexity Theory & AI Agents — A conceptual and experimental exploration of how complexity classes, quantum principles, and information theory constrain what AI agents can solve efficiently. Focuses on verification vs generation, search limits, heuristics, and implications for practical agent design.
